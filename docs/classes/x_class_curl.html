Bugfish - PHP Curl Class

This is a class designed for managing Curl requests. Please ensure that the PHP Curl module is installed and operational to use this class effectively. If you intend to enable logging through the constructor, MySQL is also required for proper functionality.
✔️ Tested
✔️ Productive - Ready
⚠️ Does use MySQL (If Logging activated)(x_class_mysqli object)
⚠️ Tables will be automatically installed
⚠️ Needs PHP Module: curl

How to use x_class_curl?

Parameters Description
last_info last curl info out of last request
Class: x_class_curl
Parameters
- $last_info (public): Holds information about the last Curl request (initialized as false).
- $logging (private): Indicates whether logging is enabled or not (initialized as false).
- $logging_settings (private): Indicates whether to log request settings (initialized as false).
- $logging_table (private): The name of the logging table (initialized as false).
- $mysql (private): An instance of the MySQL database connection (initialized as false).
Functions Results
- logging($mysql, $logging, $logging_settings, $logging_table): Initializes logging settings and creates the logging table if it doesn't exist. No direct result returned.
- create_table(): Creates the logging table in the database. Returns the result of the database query.
- xml_to_array($xml): Converts XML data to an associative array. Returns the converted array.
- xml_to_json($xml): Converts XML data to JSON format. Returns the converted JSON string.
- json_to_array($json): Converts JSON data to an associative array. Returns the converted array.
- json_to_xml($json): Converts JSON data to XML format. Returns the converted XML string.
- array_to_xml($array): Converts an associative array to XML format. Returns the converted XML string.
- array_to_json($array): Converts an associative array to JSON format. Returns the converted JSON string.
- request($url, $request, $settings): Performs a Curl request with various settings and logs it if needed. Returns the response content of the Curl request.
- download($remote, $local, $request, $settings): Downloads a file using Curl and logs the operation if needed. Returns the downloaded file content.
- upload($remote, $local, $request, $settings): Uploads a file using Curl and logs the operation if needed. Returns the response content of the Curl request.
Public Parameters
- $last_info

The $settings array is used to customize the behavior of Curl requests within the x_class_curl class. It allows you to dynamically specify various Curl options and parameters, such as user-agent strings, request headers, timeouts, and more, to tailor each HTTP request to specific requirements or scenarios. This flexibility enables you to adapt the class's functionality to a wide range of use cases without the need for extensive code modifications.
