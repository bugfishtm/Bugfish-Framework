x_class_nextcloud

Class to work with the nextcloud api and execute different operations on it. The functions will always return the result of the curl request to the nextcloud api, as it was delivered! This class works together with the x_class_curl class!
✔️ Tested
✔️ Productive - Ready

Constructor Function Description
__construct($x_class_curl, $nextcloudurl, $username, $password)Constructor with valid x_class_curl object, the nextcloud instance url, the administration username and password!
