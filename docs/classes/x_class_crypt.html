PHP Crypt Class [x_class_crypt]Documentations

This is the Documentation for the Bugfish Framework!

Documentation

The x_class_crypt class is designed for data encryption and decryption in PHP applications. It utilizes the Advanced Encryption Standard (AES) algorithm with a specific cipher block chaining (CBC) mode, specified by the chosen algorithm during class instantiation. The class offers methods to encrypt and decrypt data using the provided encryption key. This class can be used to secure sensitive data in your application by providing encryption and decryption capabilities.

Class Functions

Function Description Parameters
__construct Constructor method for initializing the object $algo (optional): The encryption algorithm to use (default is 'aes-256-cbc')
encrypt Encrypts data using the specified algorithm $data: The data to be encrypted
$key: The encryption key used for encryption
decrypt Decrypts data using the specified algorithm $data: The data to be decrypted
$key: The encryption key used for decryption
