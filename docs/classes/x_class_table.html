PHP Table Class [x_class_table]Documentations

This is the Documentation for the Bugfish Framework!



Documentation

The x_class_table class is a PHP class designed to simplify the management of database tables. It provides a set of functions for creating, editing, and deleting rows within a specified database table. The class also offers the flexibility to configure the structure of the HTML forms used for creating and editing table entries. It includes built-in CSRF protection to enhance security during data manipulation operations. Additionally, the class can generate an HTML table for displaying the contents of the database table, making it a valuable tool for building web applications that involve database interaction.

⚠️ Does use Session-Cookies
⚠️ Does need x_class_mysql object
⚠️ Does need x_class_csrf class available

Class Functions

Below you can see a list of functions and explanations on how to work with this class!

Function/Class Description Parameters
class x_class_table A PHP class for managing tables.
  • $mysql (object): The MySQL database connection.
  • $table_name (string): The name of the table.
  • $id (string): An optional identifier for the class instance.
  • $id_field (string): The name of the primary key field (default is "id").
__construct Constructor for initializing the class.
  • $mysql (object): The MySQL database connection.
  • $table_name (string): The name of the table.
  • $id (string): An optional identifier for the class instance.
  • $id_field (string): The name of the primary key field (default is "id").
exec_delete Executes a delete operation on a table row. No parameters are required.
config_rel_url Configures the relative URL for form actions.
  • $rel_url (string): The relative URL to set.
config_array Configures the arrays for creation and editing.
  • $create (array): Configuration for creating.
  • $edit (array): Configuration for editing.
exec_edit Executes an edit operation on a table row. No parameters are required.
exec_create Executes a create operation to add a new row to the table. No parameters are required.
spawn_return Generates a message box to display operation results.
  • $deleted (string): Message for deleted item.
  • $csrf (string): Message for CSRF error.
  • $edited (string): Message for edited item.
  • $created (string): Message for created item.
spawn_create Generates the form for creating a new item.
  • $button_name (string): Label for the create button.
  • $button_class (string): CSS class for the button.
spawn_edit Generates the form for editing an existing item.
  • $button_name (string): Label for the edit button.
  • $button_class (string): CSS class for the button.
spawn_table Generates the table for listing items in the database.
  • $title_array (array): Configuration for table headers.
  • $value_array (array): Data to display in the table.
  • $editing (boolean): Enable or disable editing links.
  • $deleting (boolean): Enable or disable deleting links.
  • $creating (string): Label for creating a new item link.
  • $action_column (string): Header label for action column.
  • $classes (string): Additional CSS classes for the table.
