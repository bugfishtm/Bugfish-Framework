PHP Hitcounter Class [x_class_hitcounter]Documentations

This is the Documentation for the Bugfish Framework!

Documentation

The x_class_hitcounter class is designed to manage a hit counter for tracking the number of hits and visits to specific URLs. It interacts with a MySQL database to store and retrieve hit counter data. Key features of the class include the ability to count arrivals and switches, refresh counters, prepare URLs for storage, and execute hit counting. The class also offers options for enabling or disabling hit counting and clearing GET parameters from URLs before storage. It maintains counters for arrivals, switches, and the summarized total of hits and visits for a given URL. The class uses PHP sessions to manage the state of visited URLs and store hit counter data in the MySQL database.

⚠️ Does use MySQL (x_class_mysqli object)
⚠️ Tables will be automatically installed
⚠️ Does use Session-Cookies

Class Functions

Construct Function Description
__construct($x_class_mysql, $table, $precookie = "" ) Create the Class and Create Table if not Exists with x_class_mysql object and Precookie String is optional!
Execute Function Description
execute() Executes the Counting Function for a Site if not Disabled by Config Function
Public Variables and List Function Description
switches Current Switch Value
arrivals Current Arrive Value
summarized Current Hits (Both Sum) Value
get_array() Get current list of websites registered (views) in array
Config Function Description
enabled($bool = true) Disable or Enable the Hits Check for this Page
clearget($bool = true) Clear Get Varibales in Logged URL

Example Implementation

// Include the necessary files and set up your database connection here.

// Create an instance of the x_class_hitcounter class.
$hitCounter = new x_class_hitcounter($your_db_connection, 'hit_counter_table', 'precookie_');

// You can disable hit counting if needed (e.g., for certain pages).
// $hitCounter->enabled(false);

// You can also enable or disable cleaning of the URL query string.
// $hitCounter->clearget(true);

// Execute the hit counter to track the current page view.
$hitCounter->execute();

// Retrieve the hit counts for the current page.
$switches = $hitCounter->switches;
$arrivals = $hitCounter->arrivals;
$summarized = $hitCounter->summarized;

// Display the hit counts on your web page.
echo "Switches: $switches
";
echo "Arrivals: $arrivals
";
echo "Summarized: $summarized
";

// Optionally, you can get an array of all saved referers.
// $referers = $hitCounter->get_array();
// print_r($referers);

// Close your database connection if necessary
