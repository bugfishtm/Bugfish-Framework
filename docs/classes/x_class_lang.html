Bugfish - PHP Lang Class

The x_class_lang class handles multilingual translation and localization. It provides methods for managing translations both through a MySQL database and a file-based approach. The class can initialize translation arrays, delete translation keys, add new translations, and perform translations using the loaded data. It simplifies the process of translating content for different languages and sections within an application.

How to use x_class_lang?

Function Parameters Result Explanation Example
create_table() None Executes SQL to create a table Creates a MySQL table for storing translations if it doesn't already exist. php $x_class_lang->create_table();
__construct() $mysql, $table, $lang, $section, $file_name Initializes class properties Constructor for the class. It initializes properties like database connection, table name, language, section, and file mode. If the MySQL object is provided, it checks if the table exists and creates it if not. If a file name is provided, it reads translations from a file. php $x_class_lang = new x_class_lang($mysql, $table, $lang, $section, $file_name);
init() None Initializes translation array Initializes the translation array by fetching translations from the database. It is called automatically in the constructor. Automatically called in constructor
delete($key, $lang) $key (string), $lang (optional string) Deletes a translation entry Deletes a translation entry for a specific key and language from the database. If $lang is not provided, it deletes the entry for the current language and section. php $x_class_lang->delete('example_key', 'en');
add($key, $text, $lang) $key (string), $text (string), $lang (optional string) Adds a new translation entry Adds a new translation entry to the database. If $lang is not provided, it adds it for the current language and section. php $x_class_lang->add('new_key', 'New Text', 'fr');
translate($key) $key (string) Translates a key Translates a given key. If the key exists in the translation array, it returns the translation; otherwise, it returns the key itself. php $x_class_lang->translate('example_key');
