x_class_lang

The x_class_lang class handles multilingual translation and localization. It provides methods for managing translations both through a MySQL database and a file-based approach. The class can initialize translation arrays, delete translation keys, add new translations, and perform translations using the loaded data. It simplifies the process of translating content for different languages and sections within an application.

Construct Function Description
__construct($mysql = false, $table = false, $lang = "none", $section = "none", $file_name = false) Construct the Class with x_class_mysql_object, set a lang identifier which will be loaded for the object (for translations) and a section for multi-site use support of the same table (optional), If filename is there you can load language out of a language file (# / // for comment and = as seperator)
Functions Description
delete($key, $lang = false) Delete a Translation with key. (Optional is to enter a lang string as well to delete a lang specific key, otherwhise the current loaded lang key will be deleted!
add($key, $text, $lang = false) Add a Translation with key. (Optional is to enter a lang string as well to add a lang specific key, otherwhise the current loaded lang key will be related!
translate($key) Get Translation text for current loaded language with provided key.
