PHP Redis Class [x_class_redis]Documentations

This is the Documentation for the Bugfish Framework!



Documentation

The x_class_redis class facilitates interactions with a Redis server, offering methods for connecting, adding and retrieving data, and managing keys and lists. It includes functionalities like checking connection validity, pinging the server, fetching keys, adding strings and lists, and retrieving values. This class simplifies Redis operations for caching and fast data storage.

⚠️ Needs PHP Redis Module


Class Functions

Below a list of functions and explanations on how to use them.

Constructor Description
__construct($host, $port, $pre = "") Host / Redis Port and Pre for all Keys (optional)
Functions Description
redis() Get Redis Object
valid() True if the current Redis Connction is successfull!
ping() Get Redis Ping Return
keys( $pre = false, $after = "") return $this->redis->keys($pre."*".$after);
if $pre is not set, default pre from construction will be used
add_string( $name, $value) Add String as Redis Key Name
add_list( $name, $value) Add Array as Redis (List) Key Name
get_string( $name) Get Key String
get_list($name, $start, $end) Get Key list to Array

Example Implementation

Below you can see one example implementation of this class. It does only show a short introduction on how to initialize it.

// Include the class definition or autoload it if needed

// Define your Redis server connection details
$redisHost = 'localhost'; // Redis server hostname or IP address
$redisPort = 6379; // Redis server port
$redisPrefix = 'my_prefix'; // Optional key prefix

// Create an instance of the x_class_redis class
$redisClient = new x_class_redis($redisHost, $redisPort, $redisPrefix);

// Check if the connection to Redis is successful
if ($redisClient->valid()) {
echo "Connected to Redis\n";

// Ping the Redis server
$pingResponse = $redisClient->ping();
echo "Ping Response: $pingResponse\n";

// Add a string value to Redis
$redisClient->add_string('my_key', 'Hello, Redis!');

// Retrieve the string value from Redis
$redisValue = $redisClient->get_string('my_key');
echo "Retrieved Value: $redisValue\n";

// Add items to a Redis list
$listData = ['item1', 'item2', 'item3'];
$redisClient->add_list('my_list', $listData);

// Retrieve a range of items from the Redis list
$listItems = $redisClient->get_list('my_list', 0, -1);
echo "List Items: " . implode(', ', $listItems) . "\n";
} else {
echo "Failed to connect to Redis\n";
}
