x_class_redis [Mastered]

Class to Control Redis Catching
Needs php Module: redis and redis-server linux package
Constructor Description
__construct($host, $port, $pre = "") Host / Redis Port and Pre for all Keys (optional)
Functions Description
redis() Get Redis Object
ping() Get Redis Ping Return
keys() return $this->redis->keys("*");
add_string($name, $value) Add String as Redis Key Name
add_list($name, $value) Add Array as Redis (List) Key Name
get_string($name) Get Key String
get_list($name, $start, $end) Get Key list to Array
