Bugfish - PHP Comment Class

The x_class_comment is a PHP class for managing a commenting system. It handles comment display, user voting, and submission. It interacts with a MySQL database, provides voting and comment forms, and allows comments to be shown based on filters. The class simplifies the integration of a commenting system into websites or applications.
