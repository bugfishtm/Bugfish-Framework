PHP ZIP Class [x_class_zip]Documentations

This is the Documentation for the Bugfish Framework!

Documentation

The x_class_zip class is a utility class for handling the compression and decompression of files using the ZIP format. This class provides methods to zip and unzip files, and it also supports encryption and decryption of the zip files if you provide an encryption class (x_class_crypt) and a cryptokey.

⚠️ Needs PHP Zip Module

Class Functions

Conversion Functions Description
zip($file_source, $file_destination_zip, $x_class_crypt = false, $cryptokey = false, $tempfile = false) Zip File Constructor Settings Needs Zip File Created Location $zipfile_path and Folder to Zip $folder_path
Last 3 Parameters are Optional to crypt with x_class_crypt files directory and unzip crypted files directly. Errors will be shown in php log if there are any, watch out for these if something is not working. The last parameter defines the path for the temp file. By default a tempfile will be created and deleted after the encryption/decryption process in the same folder as the destination file.
unzip($from, $to, $x_class_crypt = false, $cryptokey = false, $tempfile = false) Unzip File Constructor Settings Needs Zip File Created Location $zipfile_path and Folder to Zip $folder_path
Last 3 Parameters are Optional to crypt with x_class_crypt files directory and unzip crypted files directly. By default a tempfile will be created and deleted after the encryption/decryption process in the same folder as the destination file.

Zip a File

// Include the class definition or autoload it if needed.
// $x_class_crypt should be your cryptography class, and $cryptokey should be your encryption key (if used).

$zipper = new x_class_zip();

// Zip a file without encryption
$sourceFile = '/path/to/source/file.txt';
$destinationZip = '/path/to/destination/file.zip';
$result = $zipper->zip($sourceFile, $destinationZip);

if ($result) {
echo "File zipped successfully.";
} else {
echo "Error zipping file.";
}

Zip Encrypted

// Include the class definition or autoload it if needed.
// $x_class_crypt should be your cryptography class, and $cryptokey should be your encryption key.

$zipper = new x_class_zip();

// Zip a file with encryption
$sourceFile = '/path/to/source/file.txt';
$destinationZip = '/path/to/destination/encrypted_file.zip';
$result = $zipper->zip($sourceFile, $destinationZip, $x_class_crypt, $cryptokey);

if ($result) {
echo "File zipped and encrypted successfully.";
} else {
echo "Error zipping and encrypting file.";
}

Unzipping a File:

// Include the class definition or autoload it if needed.
// $x_class_crypt should be your cryptography class, and $cryptokey should be your decryption key (if used).

$zipper = new x_class_zip();

// Unzip a file without decryption
$sourceZip = '/path/to/source/file.zip';
$destinationFolder = '/path/to/destination/folder/';
$result = $zipper->unzip($sourceZip, $destinationFolder);

if ($result) {
echo "File unzipped successfully.";
} else {
echo "Error unzipping file.";
}

Unzipping an Encrypted File:

// Include the class definition or autoload it if needed.
// $x_class_crypt should be your cryptography class, and $cryptokey should be your decryption key.

$zipper = new x_class_zip();

// Unzip an encrypted file
$sourceZip = '/path/to/source/encrypted_file.zip';
$destinationFolder = '/path/to/destination/folder/';
$result = $zipper->unzip($sourceZip, $destinationFolder, $x_class_crypt, $cryptokey);

if ($result) {
echo "File unzipped and decrypted successfully.";
} else {
echo "Error unzipping and decrypting file.";
}
