x_class_ipbl

The x_class_ipbl class serves as a tool for IP blacklisting control. It facilitates managing and controlling access from specific IP addresses through interaction with a MySQL database. Key features of the class include the ability to check if an IP address is blocked, retrieve an array of banned IPs, unblock a specific IP, retrieve the failure counter for an IP address, raise the counter for an IP, and manage the state of blocked IPs. The class employs a MySQL database for storing IP blacklisting information and controlling access to the application.
