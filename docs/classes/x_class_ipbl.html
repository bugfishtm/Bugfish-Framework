x_class_ipbl

Class to control IP Blacklistings Needs x_class_mysql object! Tables will be auto-installed upon first run. To reset the blacklist, you should add a cronjob which deletes all entries of the related created tables which has been specified in the constructor!
✔️ Tested
✔️ Productive - Ready
