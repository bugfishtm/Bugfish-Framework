x_class_perm

Class to control simple Permissions Needs x_class_mysql object! Tables will be auto-installed upon first run.
✔️ Tested
✔️ Productive - Ready

This object can not be created by itself. It needs to be obtained by getting it from the x_class_perm function: item($ref, $permission_set = false) [See above class description]
