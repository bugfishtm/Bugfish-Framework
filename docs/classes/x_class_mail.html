PHP Mail Class [x_class_mail]Documentations

This is the Documentation for the Bugfish Framework!



Documentation

The x_class_mail class is a PHP class designed for sending emails using the PHPMailer library. It encapsulates various email-related functionalities and configurations. The class includes features such as setting up SMTP authentication, defining host details, configuring secure connections, managing email recipients (To, CC, BCC), attaching files, and handling HTML content. The class provides methods to send emails using either a simplified approach (method send) or an advanced method (mail) that allows more customization. It also supports logging email activities, allowing developers to track sent emails. The class can be used to create email sending instances (x_class_mail_item) that hold specific settings and attachments, providing a convenient way to send emails with specific configurations.

⚠️ Does use MySQL (x_class_mysqli object) (for logging)
⚠️ Tables will be automatically installed

Example Implementation

Below you can see one example implementation of this class. It does only show a short introduction on how to initialize it.

// Include the PHPMailer library and create an instance of x_class_mail
require 'path/to/PHPMailer.php';
require 'path/to/Exception.php';
require 'path/to/x_class_mail.php';

$mail = new x_class_mail('your_smtp_server.com', 587, 'tls', 'your_username', 'your_password', 'your_email@example.com', 'Your Name');

// Set additional configurations (optional)
$mail->charset('UTF-8');
$mail->encoding('base64');
$mail->allow_insecure_ssl_connections(false);
$mail->smtpdebuglevel(0);
$mail->all_default_html(false);
$mail->change_default_template('', ''); // You can specify custom header and footer here
$mail->initReplyTo('reply@example.com', 'Reply Name');
$mail->test_mode(false);

// Create an instance of x_class_mail_item for easier email configuration
$email = $mail->object();

// Add recipients
$email->add_receiver('recipient1@example.com', 'Recipient 1');
$email->add_receiver('recipient2@example.com', 'Recipient 2');

// Add CC and BCC recipients (optional)
$email->add_cc('cc@example.com', 'CC Recipient');
$email->add_bcc('bcc@example.com', 'BCC Recipient');

// Add attachments (optional)
$email->add_attachment('path/to/attachment1.pdf', 'Attachment 1');
$email->add_attachment('path/to/attachment2.jpg', 'Attachment 2');

// Set additional settings (optional)
$email->add_setting('header', 'Custom Header');
$email->add_setting('footer', 'Custom Footer');

// Send the email
$subject = 'Your Subject';
$content = 'Email content goes here. You can use HTML if HTML mode is enabled.';
if ($email->send($subject, $content)) {
echo 'Email sent successfully!';
} else {
echo 'Email sending failed. Error: ' . $mail->last_info();
}
