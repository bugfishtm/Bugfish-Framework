PHP Log Class [x_class_log]Documentations

This is the Documentation for the Bugfish Framework!



Documentation

The x_class_log class is designed to manage logging and recording various types of messages, notifications, warnings, and errors to a MySQL database. It offers an organized way to keep track of system events, errors, and important information. The class interacts with the provided MySQL connection and table to store log entries.

⚠️ Does use MySQL (x_class_mysqli object)
⚠️ Tables will be automatically installed

Table Structure

Table for this class to store logging activities.

Column Name Data Type Constraints Description
id int(10) NOT NULL, AUTO_INCREMENT Identificator
type int(10) DEFAULT '0' 0 - Unspecified, 1 - Error, 2 - Warning, 3 - Notification
message text Logged Text
section VARCHAR(128) NULL Logged Category
creation datetime DEFAULT CURRENT_TIMESTAMP Creation Date (Auto-Set on Insert)

Class Functions

Below a list of functions and explanations on how to use them.

Constructor Function Description
__construct($x_class_mysql, $tablename, $section = "") Construct with an x_class_mysql object! Tablename for Table Name which should be created and Section is optional for multi sites!
Misc Functions Description
get_array() Get complete Array out of Log Table with Log Entries
list_get($limit = 50) Get Log list as Array
list_flush_section() Mass Delete Entries from Log Table for the current Classes Section and set Auto Increment Counter Back to lowest Possible
list_flush() Mass Delete Entries from Log Table and setAuto Increment Counter to 1
Message to Log Functions Description
message($message, $type = 3) Write an Message to Log with own Type Number
post($message, $type = 3) Alias
send($message, $type = 3) Alias
write($message, $type = 3) Alias
Notify Functions Description
notify($message) Write an Type 3 Message to Log
info($message) Alias
Warn Functions Description
warning($message) Write an Type 2 Message to Log
warn($message) Alias
Error Functions Description
error($message) Write an Type 1 Message to Log
failure($message) Alias
err($message) Alias
fail($message) Alias

Example Implementation

Below you can see one example implementation of this class. It does only show a short introduction on how to initialize it.

// Assuming you have a MySQL database connection already established $mysql = new mysqli('localhost', 'username', 'password', 'database_name'); // Include the x_class_log class definition here // Create an instance of the x_class_log class $log = new x_class_log($mysql, 'log_table', 'example_section'); // Log some messages $log->error('This is an error message'); $log->warning('This is a warning message'); $log->notify('This is a notification'); // Retrieve and display the log entries for the 'example_section' $logEntries = $log->list_get(10); // Get the last 10 log entries foreach ($logEntries as $entry) { echo 'Type: ' . $entry['type'] . '-'; echo 'Message: ' . $entry['message'] . '-'; echo 'Creation Date: ' . $entry['creation'] . '-'; } // Clear the log entries for the 'example_section' $log->list_flush_section();
