PHP Referer Class [x_class_referer]Documentations

This is the Documentation for the Bugfish Framework!



Documentation

The x_class_referer class is designed to manage and control referers in a PHP application. It allows tracking and logging of incoming referer URLs. This class interacts with a MySQL database to store and retrieve referer information. Key features include enabling or disabling referer tracking, preparing URLs for database storage, retrieving saved referers as an array, and executing referer tracking functionality. The class ensures that referer URLs are appropriately processed and stored for analysis.

⚠️ Does use Session-Cookies
⚠️ Does use MySQL (x_class_mysqli object)
⚠️ Tables will be automatically installed


Table Structure

Table below is created/used by this class to save referer Informations!

Column Name Data Type Constraints Description
id int(10) NOT NULL, AUTO_INCREMENT Identificator
full_url varchar(512) NOT NULL, DEFAULT '0' Related Referer
hits int(10) NOT NULL, DEFAULT '0' Counted Hits
section varchar(64) NOT NULL, DEFAULT '' Related Section
creation datetime DEFAULT CURRENT_TIMESTAMP Creation Date (Auto-Set on Insert)
modification datetime DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP Modification Date (Auto-Set on Change)

Class Functions

Below a list of functions and explanations on how to use them.

Function Description
__construct($x_class_mysqlvar, $table, $refurlnowww) Constructor with x_mysql_class object and table name and URL without WWW and HTTP/S
enabled($bool = true) Disable / Enable for this Page
execute($section = "") Execute the Referer Related Functions if not Disabled with enabed = true function! Provide $section Variable for Multi Site of Needed
get_array() Get current Referer URLS saved in Table out in Array!

Example Implementation

Below you can see one example implementation of this class. It does only show a short introduction on how to initialize it.

// Include your database connection code here.
// For example, you might have something like:
// $mysql = new mysqli('localhost', 'username', 'password', 'database_name');

// Include the x_class_referer class here.
// Make sure you include it after setting up the database connection.
// require_once('x_class_referer.php');

// Instantiate the x_class_referer class.
// Pass the database connection, table name, and whether to include 'www' in the referrer URL.
$refererTracker = new x_class_referer($mysql, 'referrer_data', true);

// Enable or disable the tracking as needed.
// $refererTracker->enabled(true); // Enable tracking

// Execute the tracking when needed, typically on the page where you want to track referrals.
$refererTracker->execute();

// Retrieve the current saved referers in an array.
$referers = $refererTracker->get_array();

// Display or process the $referers array as needed.
foreach ($referers as $referer) {
echo "Referer: " . $referer['full_url'] . " - Hits: " . $referer['hits'] . "
";
}

// Don't forget to close your database connection when you're done.
// $mysql->close();
