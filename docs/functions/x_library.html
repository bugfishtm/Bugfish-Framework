x_library

Embark on a journey through an indispensable collection of PHP functions meticulously crafted for seamless integration across a multitude of projects. Witness firsthand how these functions expedite and refine coding processes within web development. Comprehensive documentation for each function is presented in the following list. To seamlessly incorporate these functions, the PHP files can be effortlessly included within the designated _functions subfolder nestled within the overarching _framework folder. This strategic approach not only optimizes coding efficiency but also transforms the landscape of web project performance.

Thumbnail Function Name Description
x_thumbnail This function creates a thumbnail image from a URL and saves it to a file. It takes the following parameters:
- $url: The URL of the image to create a thumbnail from.
- $filename: The filename for the saved thumbnail.
- $width: The width of the thumbnail.
- $height: The height of the thumbnail (or true to maintain aspect ratio).
x_thumbnail_save This function creates a thumbnail image from a URL and optionally saves it to a file. It takes the following parameters:
- $url: The URL of the image to create a thumbnail from.
- $save_path: An optional path to save the thumbnail file.
- $width: The width of the thumbnail.
- $height: The height of the thumbnail (or true to maintain aspect ratio).
Library Function Name Description Parameter
x_firstimagetext $text, $all = false - $text: Input text containing HTML.
- $all: Whether to return all image tags or just the first one. Parses HTML text to extract image tags and returns either the first image tag or all of them, depending on the $all parameter.
x_connection_check $host, $port, $timeout = 1 - $host: Hostname or IP address to check the connection to.
- $port: Port number to check.
- $timeout: Timeout duration in seconds. Checks if a connection can be established to the specified host and port within the given timeout. Returns true if successful, false otherwise.
x_inCLI None Checks if the script is running in a Command Line Interface (CLI) environment and returns true if it is, false otherwise.
x_rmdir $dir - $dir: Directory path to be deleted recursively. Recursively deletes all files and subdirectories within the specified directory.
x_html_redirect $url, $seconds = 0 - $url: URL to redirect to.
- $seconds: Number of seconds before the redirection occurs. Generates an HTML meta tag for redirection to the specified URL after the specified number of seconds.
Validation Functions Description
x_isset This function checks if a value is not null and has a length greater than 1.
x_imgValid This function checks if a given string represents a valid image URL.
x_contains_cyrillic This function checks if a string contains Cyrillic characters.
x_contains_bad_word This function checks if a string contains bad words based on a filter.
x_contains_url This function checks if a string contains a URL.
x_hsc This function is a simple replacement for htmlspecialchars.
x_getint This function retrieves a GET value as an integer if it is numeric.
x_postint This function retrieves a POST value as an integer if it is numeric.
x_get This function retrieves a GET value.
x_post This function retrieves a POST value.
x_datediff_before This function checks if one date is a certain length of time before another date.
Button Functions Description Parameter
x_executionButton $db, $name, $url, $query, $get, $msgerr = "Fehler!", $msgok = "Erfolgreich!", $break = false, $style = "" - $db: Database connection object.
- $name: Button name.
- $url: URL to execute the action.
- $query: SQL query to execute.
- $get: GET parameter to trigger the action.
- $msgerr: Error message.
- $msgok: Success message.
- $break: Whether to add a line break after the button.
- $style: CSS style for the button.
x_button $name, $url, $break = false, $style = "", $reacttourl = true - $name: Button name.
- $url: URL to navigate to.
- $break: Whether to add a line break after the button.
- $style: CSS style for the button.
- $reacttourl: Whether to style the button differently if it matches the current URL.
EventBox Functions Description Parameter
x_eventBoxPrep $text, $type = "x", $precookie = "", $morecss = "", $buttontext = "X", $imgok = false, $imgfail = false, $imgwarn = false, $imgelse = false - $text: Text to display in the event box.
- $type: Type of event box (ok, warning, error, etc.).
- $precookie: Session variable name for storing event box content.
- $morecss: Additional CSS styles for the event box.
- $buttontext: Text for the close button.
- $imgok, $imgfail, $imgwarn, $imgelse: Optional images to display in the event box.
x_eventBoxShow $precookie = "" - $precookie: Session variable name for event box content. Displays the event box content and clears the session variable.
x_eventBoxSet $precookie = "" - $precookie: Session variable name for event box content. Checks if the event box content is set in the session.
x_eventBoxSkip $precookie = "" - $precookie: Session variable name for event box content. Sets a flag to skip displaying the event box.
CookieBanner Functions Description Parameter
x_cookieBanner_Pre $precookie = "", $redirect = true - $precookie: Session variable name for tracking if the cookie banner has been accepted.
- $redirect: Whether to redirect after accepting the cookie banner.
x_cookieBanner $precookie = "", $use_post = false, $text = false, $url_cookies = "" - $precookie: Session variable name for tracking if the cookie banner has been accepted.
- $use_post: Whether to use POST method for accepting the banner.
- $text: Text to display in the cookie banner.
- $url_cookies: URL to the cookies policy.
RSS Functions Description Parameter
x_rss_list $urltemp, $defaultcover, $limit = 25 - $urltemp: URL of the RSS feed to parse.
- $defaultcover: Default cover image URL for RSS items without images.
- $limit: Maximum number of RSS items to display. Parses an RSS feed and displays a list of items with their titles, links, dates, and images.
x_rss_array $urltemp - $urltemp: URL of the RSS feed to parse. Parses an RSS feed and returns an array of items with their titles, links, dates, and images.
Captcha Functions Description Parameter
x_captcha $preecookie = "", $width = 550, $height = 250, $square_count = 5, $eclipse_count = 5, $color_ar = false, $font = "", $code = "" - $preecookie: Session variable name for storing the captcha code.
- $width: Width of the captcha image.
- $height: Height of the captcha image.
- $square_count: Number of squares in the captcha.
- $eclipse_count: Number of ellipses in the captcha.
- $color_ar: An array of colors for various elements.
- $font: Font file for text in the captcha.
- $code: The captcha code to display.
x_captcha_key $preecookie = "" - $preecookie: Session variable name for the captcha code. Returns the captcha code stored in the session.
Search Functions Description
x_search This function performs a database search using MySQL. It takes the following parameters:
- $mysql: The MySQL database connection.
- $table: The name of the table to search in.
- $search_fields: An array of fields to search in.
- $get_fields: An array of fields to retrieve in the results.
- $search_string: The search string.
- $uniqueref: The unique reference field (default is "id").
Table Functions Description
x_table_div This function generates an HTML table from an array of data. It takes the following parameters:
- $array: The array of data to display in the table.
- $titlelist: An array of column titles.
- $alignarray: An optional array specifying the alignment of each column.
- $percentarray: An optional array specifying the width of each column as a percentage.
- $title: An optional title for the table.
x_table_complex This function generates a complex HTML table with a search filter form. It takes the following parameters:
- $array: The array of data to display in the table.
- $titlelist: An array of column titles.
- $formid: An optional form ID for the search filter form.
- $alignarray: An optional array specifying the alignment of each column.
x_table_simple This function generates a simple HTML table from an array of data. It takes the following parameters:
- $array: The array of data to display in the table.
- $titlelist: An array of column titles.
- $tableid: An optional table ID.
- $alignarray: An optional array specifying the alignment of each column.
