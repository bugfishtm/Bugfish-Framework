This is the Documentation for the Bugfish Framework!



xjs_library.js

Explore a comprehensive array of JavaScript functions designed for universal integration across diverse projects, enhancing coding speed and efficiency within web development. Detailed documentation for each function is provided in the list below. To seamlessly incorporate these functions, simply include the JavaScript file within your project's designated _javascript subfolder, located within the central _framework folder. This strategic implementation promises to optimize your coding process and elevate your web project's performance. Simplify your coding process with our comprehensive library of pre-built functions and modules, carefully crafted to expedite the creation of dynamic, interactive web applications. Say farewell to development complexities and embrace a more streamlined programming experience. You can find the Javascript Function file in: /_framework/javascript/*.js!


Function Explanation Parameters
xjs_get(param) Extracts the value of a specific GET parameter from the URL. param (String): The name of the GET parameter to retrieve.
xjs_in_url(param) Checks if a given string exists within the current URL. param (String): The string to search for within the URL.
xjs_hide_id(id) Hides an HTML element with a specified ID by modifying its CSS property. id (Object or String): The ID of the HTML element to hide.
xjs_show_id(id) Shows a hidden HTML element with a specified ID by altering its CSS property. id (Object or String): The ID of the HTML element to show.
xjs_toggle_id(id) Toggles the visibility of an HTML element with a specified ID. id (Object or String): The ID of the HTML element to toggle.
xjs_is_email(email) Checks if a given string represents a valid email address using a regular expression pattern. email (String): The email address to validate.
xjs_popup(var_text, var_entrie) Dynamically creates a customizable popup element with an optional close button. var_text (String): The content to display in the popup.
var_entrie (String, optional): The text for the close button (default is "Close").
xjs_genkey(length, charset) Generates random passwords of a specified length using characters from a character set. length (Number, optional): The length of the generated password (default is 12).
charset (String, optional): The character set for password generation (default includes letters and digits).
