xfpe_library

Introducing a CSS Classes File tailored for swift design implementation. Appending "_f" to a class enhances its importance. This resource encompasses style definitions for XML and HTML classes, expediting and refining styling with precision. To begin, include the required file in the "_css" directory within the framework's folder. Streamline your design process with efficiency and accuracy.

Add _f to end of a class to make it important!
