Margin Classes

Padding / Margin Class Description Possible Values Max Value of N
.xfpe_nopadding Removes padding from all sides (sets to 0px)
.xfpe_nopaddingtop Removes padding from the top (sets padding-top to 0px)
.xfpe_nopaddingbottom Removes padding from the bottom (sets padding-bottom to 0px)
.xfpe_nopaddingleft Removes padding from the left (sets padding-left to 0px)
.xfpe_nopaddingright Removes padding from the right (sets padding-right to 0px)
.xfpe_paddingleft{N}px Sets the left padding of the element to N pixels. N: 5, 10, 15, ... 150
.xfpe_paddingright{N}px Sets the right padding of the element to N pixels. N: 5, 10, 15, ... 150
.xfpe_paddingleftm{N}px Sets a negative left padding of the element to N pixels. N: 5, 10, 15, ... 150
.xfpe_paddingrightm{N}px Sets a negative right padding of the element to N pixels. N: 5, 10, 15, ... 150
.xfpe_paddingbottom{N}px Sets padding-bottom to a specific value in pixels (Xpx) N: 5, 10, 15, ... 150
.xfpe_paddingbottomm{N}px Sets negative padding-bottom to a specific value in pixels (-Xpx) N: 5, 10, 15, ... 150
.xfpe_paddingtop{N}px Sets padding-top to a specific value in pixels (Xpx) N: 5, 10, 15, ... 150
.xfpe_paddingtopm{N}px Sets negative padding-top to a specific value in pixels (-Xpx) N: 5, 10, 15, ... 150
.xfpe_marginauto Sets margin to "auto" for all sides
.xfpe_nomargin Removes margin from all sides (sets to 0px)
.xfpe_nomargintop Removes margin from the top (sets margin-top to 0px)
.xfpe_nomarginbottom Removes margin from the bottom (sets margin-bottom to 0px)
.xfpe_nomarginleft Removes margin from the left (sets margin-left to 0px)
.xfpe_nomarginright Removes margin from the right (sets margin-right to 0px)
.xfpe_marginleft{N}px Sets the left margin of the element to N pixels. N: 5, 10, 15, ... 150
.xfpe_marginright{N}px Sets the right margin of the element to N pixels. N: 5, 10, 15, ... 150
.xfpe_marginleftm{N}px Sets a negative left margin of the element to N pixels. N: 5, 10, 15, ... 150
.xfpe_marginrightm{N}px Sets a negative right margin of the element to N pixels. N: 5, 10, 15, ... 150
.xfpe_margintop{N}px Sets the top margin of the element to N pixels. N: 5, 10, 15, ... 150
.xfpe_margintopm{N}px Sets negative margin-top to a specific value in pixels (-Xpx) N: 5, 10, 15, ... 150
.xfpe_marginbottom{N}px Sets margin-bottom to a specific value in pixels (Xpx) N: 5, 10, 15, ... 150
.xfpe_marginbottomm{N}px Sets negative margin-bottom to a specific value in pixels (-Xpx) N: 5, 10, 15, ... 150

Width/Height Classes

Width / Height Class Description Possible Values Max Value of N
.xfpe_width{N}px Sets the element's width to N pixels. N: 25, 50, 75, 100, ... 1000
.xfpe_height{N}px Sets the element's height to N pixels. N: 25, 50, 75, 100, ... 1000
.xfpe_width{N}pct Sets the element's width to N pixels. N: 25, 50, 75, 100, ... 100
.xfpe_height{N}pct Sets the element's height to N pixels. N: 25, 50, 75, 100, ... 100
.xfpe_maxwidth{N}pct Sets the element's width to N percentage of parent. N: 25, 50, 100 100
.xfpe_maxwidth{N}px Sets the maximum width of the element to N pixels. N: 50, 100, 150, ... 1000
.xfpe_minwidth{N}px Sets the minimum width of the element to N pixels. N: 50, 100, 150, ... 1000
.xfpe_minwidth{N}pct Sets the element's width to N percentage of parent. N: 25, 50, 100 100
.xfpe_maxheight{N}px Sets the maximum height of the element to N pixels. N: 50, 100, 150, ... 1000
.xfpe_minheight{N}px Sets the minimum height of the element to N pixels. N: 50, 100, 150, ... 1000
.xfpe_maxheight{N}pct Sets the maximum height of the element to N pixels. N: 25, 50, 100, ... 100
.xfpe_minheight{N}pct Sets the minimum height of the element to N pixels. N: 25, 50, 100, ... 100

Display Classes

Display Class Description
.xfpe_dispinlineblock Sets the element's display property to inline-block.
.xfpe_dispinline Sets the element's display property to inline.
.xfpe_dispblock Sets the element's display property to block.
.xfpe_dispnone Sets the element's display property to none (hides the element).
.xfpe_dispflex Sets the element's display property to flex (enables flexbox layout).
.xfpe_dispgrid Sets the element's display property to grid (enables grid layout).
.xfpe_dispinlineflex Sets the element's display property to inline-flex (enables inline flexbox layout).
.xfpe_dispinlinegrid Sets the element's display property to inline-grid (enables inline grid layout).

Float Classes

Float Class Description
.xfpe_floatleft Sets the element's float property to left.
.xfpe_floatright Sets the element's float property to right.
.xfpe_floatnone Clears any previous float property.
.xfpe_floatinherit Inherits the float property from the parent element.
.xfpe_floatinitial Sets the float property to its default value.
.xfpe_floatunset Resets the float property to its inherited value.

Overflow Classes

Overflow Class Description
.xfpe_overflowhidden Sets the element's overflow property to hidden.
.xfpe_overflowscroll Sets the element's overflow property to scroll.
.xfpe_overflowvisible Sets the element's overflow property to visible.

Text Classes

Text Class Name Explanation
.xfpe_textbreakall Breaks text at any character, preserving space
.xfpe_textnowrap Prevents text from wrapping to the next line
.xfpe_textuppercase Converts text to uppercase
.xfpe_textlowercase Converts text to lowercase
.xfpe_textcapitalize Capitalizes the first letter of each word

Align Classes Name Explanation

Cursor Classes

Cursor Class Name Explanation
.xfpe_cursorpointer Changes the cursor to a pointer hand
.xfpe_cursordefault Sets the default cursor (usually an arrow)
.xfpe_cursorauto Automatically sets the cursor based on the element
.xfpe_cursorcrosshair Sets the cursor to a crosshair
.xfpe_cursorhelp Sets the cursor to a help cursor (question mark)
.xfpe_cursormove Sets the cursor to a move cursor
.xfpe_cursornotallowed Sets the cursor to a not-allowed cursor
.xfpe_cursorprogress Sets the cursor to a progress cursor
.xfpe_cursortext Sets the cursor to a text cursor (I-beam)
.xfpe_cursorwait Sets the cursor to a wait cursor (hourglass)
.xfpe_cursoralias Sets the cursor to an alias cursor
.xfpe_cursorcopy Sets the cursor to a copy cursor
.xfpe_cursorzoomin Sets the cursor to a zoom-in cursor
.xfpe_cursorzoomout Sets the cursor to a zoom-out cursor
.xfpe_cursorgrab Sets the cursor to a grab cursor
.xfpe_cursorgrabbing Sets the cursor to a grabbing cursor

Border Class Name Explanation
